# Weather App

## How to open:

In the project directory, you can run:

### `npm install`
### `npm start`

Or you can open project on:

[Github Pages](https://martinljs.github.io/Weather-App/)

### Environment:

React
Typescript
Redux(thunk, devtools, react-redux)

- usage of ts and redux
- half of typisation
- styles for desktop
